# Tres en Raya

Juego clásico de Tres en Raya (Tic-Tac-Toe) construido con Next.js, TypeScript y Tailwind CSS.

## Requisitos

- Node.js 18+
- npm

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Producción

```bash
npm run build
npm start
```

## Con PM2

```bash
npm run build
pm2 start ecosystem.config.js
```

El proyecto se sirve en `/tres-en-raya`.
